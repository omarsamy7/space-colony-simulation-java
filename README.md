# Text-Based Space Colony Simulation (Java)

## Overview
A text-based simulation developed in Java that models the management of a space colony through resource production, conversion, consumption, and exploration. The project uses object-oriented design to organise the simulation into parser, engine, and model components.

## Features
- Text-based command system for interacting with the simulation
- Resource production, conversion, and consumption mechanics
- Colony building and entity management
- Crew assignment and population management
- Map exploration and movement system
- Save and load functionality
- Text-based graphing of resource history
- Modular object-oriented architecture

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Command-Line Interface (CLI)
- File Handling

## Project Structure
- `src/org/uob/a2/Main.java` — entry point for the simulation
- `src/org/uob/a2/engine/` — simulation engine, context, state, and core abstractions
- `src/org/uob/a2/model/` — producers, converters, consumers, and resource types
- `src/org/uob/a2/parser/` — command parsing and user interaction commands

## Purpose
This project was developed to strengthen understanding of object-oriented programming, simulation design, command parsing, file handling, and modular software structure in Java.
